# PROMPT MAESTRO — ERP ZENITH ## Sistema ERP Profesional para Venezuela | Versión 6.0 | Marzo 2025 Este documento define la arquitectura, stack y reglas de negocio para ERP ZENITH. ### Stack Técnico - **Frontend:** Next.js 14, TypeScript 5, TailwindCSS, Framer Motion, Recharts, shadcn/ui. - **Backend:** NestJS 10, Prisma ORM, PostgreSQL (Neon). - **Diseño:** Glassmorphism + Dark Mode, Tipografía League Spartan. - **Integraciones:** IA (Gemini, Groq, HF, Ollama), n8n (10 flujos). - **Cumplimiento:** LOTTT, IVA 16%, ISLR, SENIAT, BCV. ### Módulos Activos - **Operativo:** Inventario, Compras, Producción, Mantenimiento, Calidad, Flota. - **Comercial:** Ventas, POS, Marketing. - **RRHH:** Empleados, Nómina (LOTTT), Asistencia. - **Configuración:** Sistema, Seguridad, Usuarios, Tablas. - **Reportes:** Dashboard General, BI Avanzado. ### Reglas Críticas - Segregación por roles (RBAC). - Multimoneda (VES/USD) con tasa BCV. - Auditoría total. - Cálculos con Decimal.js. - Deploy en Railway (Backend) y Vercel (Frontend).
