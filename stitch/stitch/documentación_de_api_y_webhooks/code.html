Listar Clientes

Devuelve una lista paginada de clientes. Puedes filtrar los resultados por nombre, email o estado. Incluye metadatos de paginación en la respuesta.

GET
https://api.erp-saas.com/v1/clients

Parámetros de consulta

Campo Tipo Descripción
page integer Número de página para la paginación. Default: 1
limit integer Cantidad de registros por página. Default: 15, Max: 100
search string Término de búsqueda para filtrar por nombre o email.
status enum Filtrar por estado: active, inactive.

Atributos de respuesta

El objeto cliente contiene los siguientes campos:

  • id Identificador único del cliente (UUID).
  • name Nombre completo o Razón Social.
  • tax_id Número de identificación fiscal (RFC/RUT/DNI).
  • balance Saldo actual en la moneda base.
Ejemplo de Código
Laravel Http Client 
use Illuminate\Support\Facades\Http;

// Obtener token de tu configuración
$token = '1|laravel_sanctum_token...';

$response = Http::withToken($token)
    ->get('https://api.erp-saas.com/v1/clients', [
        'page' => 1,
        'limit' => 15,
        'search' => 'Empresa'
    ]);

return $response->json();
